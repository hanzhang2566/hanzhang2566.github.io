门面模式

发表于 2023-04-29 本文字数： 357 阅读时长 ≈ 1 分钟
DATE CHANGELOG
2023年4月29日 init

模式描述

门面模式(Facade)模式，为子系统提供一组统一的接口，定义一组高层接口让子系统更易用。

需求

使用门面模式来实现新闻发言人。

UML 类图

classDiagram

class Statable {
	<<interface>>
	domestic() void
	foreign() void
}

class Spokesman {
	domestic() void
	foreign() void
}

Statable <|.. Spokesman

class Commerce {
	state() void
}

class Defense {
	state() void
}

class Education {
	state() void
}

class Foreign {
	state() void
}

Spokesman o-- Commerce
Spokesman o-- Defense
Spokesman o-- Education
Spokesman o-- Foreign

应用场景

  • 解决易用性问题。例如 Linux 系统提供的系统调用函数等；
  • 解决性能问题。
  • 解决分布式事务问题。

特点

门面模式是通过门面接口提供了对封装底层组合功能的封装。
